# Task 2.3: Update Trigger

## Context

Implement `UpdateTrigger()` to modify an existing trigger's configuration.

Reference: [plan.md](../plan.md) § 3.2.

## Steps

1. **Add `UpdateTrigger(trigger *TriggerEntry)`**:
   - Validate trigger config (same as Create)
   - Find the trigger in the `.fp` file
   - Replace HCL block with new definition
   - Preserve file formatting

2. **Add trigger block replacement logic**:
   - Parse file into lines/tokens
   - Find start/end of trigger block by name
   - Replace with new HCL
   - Write back atomically

3. **Add validation**:
   - Check trigger exists before update
   - Validate new config

4. **Handle workspace changes**:
   - If workspace changes, move trigger to different file if needed
