# Task 5.2: Flow Triggers List

## Context

Create a dedicated component to list triggers for a specific flow in the flow panel.

Reference: [plan.md](../plan.md) § 5.

## Steps

1. **Create `ui/src/components/Flows/FlowTriggersList.tsx`**:
   - Takes `flowId` prop
   - Fetches triggers from `/api/flows/{flowId}/triggers`
   - Displays in compact list

2. **Add inline actions**:
   - Quick edit (opens form)
   - Quick delete (with confirmation)
   - Trigger test (runs test execution)

3. **Add status column**:
   - Trigger status badge
   - Last run timestamp
   - Error message if applicable

4. **Add trigger details on hover**:
   - Tooltip shows full trigger config
   - Pipeline reference shown
   - Cron expression shown

5. **Add sorting**:
   - Sort by name, type, status
