# Task 5.1: Trigger Tab in Flow Editor

## Context

Add a "Triggers" tab to the flow editor panel, showing triggers associated with the current flow.

Reference: [plan.md](../plan.md) § 5.

## Steps

1. **Expand FlowPanel component**:
   - Find existing flow panel (tabs: Details, Steps, etc.)
   - Add "Triggers" tab

2. **Add TriggerList to FlowPanel**:
   - Show triggers for current flow only
   - Link to full Triggers page
   - Show triggering status

3. **Add "Add Trigger" button**:
   - Only shows if no triggers exist OR allow multiple
   - Opens TriggerForm dialog with flow pre-selected

4. **Display trigger overview**:
   - Trigger count badge
   - Types breakdown (icons/colors)
   - Last run info if available
