# Task 1.2: Phase 1 — Model

**Task ID**: 1.2  
**Phase**: 1 — Schema Registry Core (API)  
**Prerequisites**: 1.1 (Config)

## Context

Create the schema model structs for API responses. `SchemaEntry` is the config struct (in config package). `SchemaResource` is the API response shape — used by handlers and returned as JSON.

Reference: [plan.md](../plan.md) § 6.2.

## Steps

1. **Create** `api/internal/model/schema.go`

2. **Define SchemaResource** (API response):
   ```go
   type SchemaResource struct {
       ID      string            `json:"id"`
       Label   string            `json:"label"`
       Type    string            `json:"type"`
       Source  SchemaSource      `json:"source"`
   }

   type SchemaSource struct {
       Path string `json:"path"`
   }
   ```

3. **Add doc comments** for exported types and fields (per project guidelines)

## Files

| Path | Action |
|------|--------|
| `api/internal/model/schema.go` | Create |

## Acceptance Criteria

- [ ] `SchemaResource` has `id`, `label`, `type`, `source.path` in JSON
- [ ] Doc comments on exported types
- [ ] No circular imports (model must not import config or service)
