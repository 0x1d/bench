# Task 2.5: Test Trigger

## Context

Implement `TestTrigger()` to simulate trigger execution (e.g., test a webhook trigger).

Reference: [plan.md](../plan.md) § 3.2.

## Steps

1. **Add `TestTrigger(flowID, triggerID string, payload map[string]any)`**:
   - Resolve trigger's pipeline reference
   - Collect required params for that pipeline
   - Build run request with test payload
   - Call Flowpipe API to execute pipeline

2. **Add webhook payload generation**:
   - For webhook-type triggers, generate test payload
   - Include timestamp, trigger metadata
   - Simulate webhook request body

3. **Add execution tracking**:
   - Return execution ID from Flowpipe
   - Allow checking execution status

4. **Handle errors**:
   - Pipeline not found
   - Invalid payload for pipeline params
   - Flowpipe connection errors
