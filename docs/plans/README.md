# Implementation Plans

This folder contains structured implementation plans for Bench features. Each plan has:

- **README.md** — Overview, current status, quick links
- **plan.md** — Full design document
- **TASKS.md** — Task breakdown with checkboxes and status

## Task Status

Tasks use markdown checkboxes:

- `[ ]` — Not started
- `[x]` — Done

Phase status: `TODO` | `IN_PROGRESS` | `DONE`
