# Task 5.3: Inline Trigger Editor

## Context

Enable creating/editing triggers directly within the flow editor panel (no separate page needed).

Reference: [plan.md](../plan.md) § 5.

## Steps

1. **Expand FlowPanel trigger section**:
   - Show trigger list
   - "Add Trigger" button adds inline form
   - Inline form is simpler (fewer options)

2. **Add inline form for simple triggers**:
   - Trigger type selector
   - selecting the flow (pre-populated)
   - Basic config fields

3. **Edit inline (or modal)**:
   - Click edit opens TriggerForm dialog
   - Full config available
   - Save(update) returns to list

4. **Delete inline confirmation**:
   - Click delete shows toast confirmation
   - Animations for smooth UX

5. **Add trigger preview**:
   - Show HCL preview of trigger
   - Only for admins or advanced mode
