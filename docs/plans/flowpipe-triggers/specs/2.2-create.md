# Task 2.2: Create Trigger

## Context

Implement `CreateTrigger()` to add a new trigger to a flow's `.fp` file.

Reference: [plan.md](../plan.md) § 3.2.

## Steps

1. **Add `CreateTrigger(trigger *TriggerEntry)`**:
   - Validate trigger config
   - Find or create the flow's `.fp` file
   - Parse existing trigger blocks (if file exists)
   - Generate new HCL trigger block
   - Append to file

2. **Add HCL trigger block generator**:
   - Generate HCL like:
     ```hcl
     trigger "schedule" "daily_report" {
       description = "Run daily"
       pipeline    = pipeline.daily_report
       cron        = "0 9 * * *"
     }
     ```
   - Handle all trigger types
   - Properly escape strings

3. **Add uniqueness check**:
   - Check if trigger ID already exists in flow
   - Return error if duplicate

4. **Update connections.fpc**:
   - If trigger references a pipeline with database connections
   - Ensure connections for those DBs are in connections.fpc

5. **Test trigger creation**:
   - Create valid schedule trigger
   - Create valid webhook trigger
   - Verify `.fp` file is updated
